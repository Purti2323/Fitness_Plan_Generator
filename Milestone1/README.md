# Milestone 1 - Front-End Development

## Objective
Develop a user-friendly fitness profile form and implement BMI calculation logic.

## Functional Requirements Implemented
- Converted height from cm to meters
- BMI = weight (kg) / (height in meters)^2
- Rounded BMI to 2 decimal places
- Classified BMI into:
  - Underweight
  - Normal
  - Overweight
  - Obese
- Input validation

## Technologies Used
- Python
- Streamlit
- Hugging Face Spaces
